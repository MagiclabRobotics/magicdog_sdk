# MagicDog SDK C++ API - 机器人主控服务

> 提供机器人系统主控制器，通过 MagicRobot 可以进行资源初始化、管理通信连接、访问各子控制器如运动控制器、音频控制器、状态监控以及传感器控制器等。

## 接口定义
<code>MagicRobot</code> 是机器人系统的统一入口类。
